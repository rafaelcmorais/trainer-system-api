# Trainer System API

Backend API for managing personal trainer operations, including:

- Student management
- Workout planning
- Training sessions
- Progress tracking

## Tech Stack

- Node.js
- Express
- PostgreSQL

## Features

- RESTful API
- Modular architecture (routes, controllers, services)
- Scalable structure

## Project Structure

src/
routes/
controllers/
services/
database/
middlewares/

```bash
